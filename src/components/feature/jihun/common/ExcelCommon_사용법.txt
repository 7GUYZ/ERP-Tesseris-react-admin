# ExcelCommon.jsx 사용법

## 주요 특징
- 클라이언트 사이드 Excel 파일 생성
- 대용량 데이터 처리 지원
- 페이징 데이터 처리
- 새로운 엑셀 다운로드 API 지원

## 기본 사용법

### 1. import
```javascript
import { downloadExcel } from '../../../../components/feature/jihun/common/ExcelCommon';
```

### 2. 기본 다운로드
```javascript
const data = [
  { '이름': '홍길동', '나이': 30 },
  { '이름': '김철수', '나이': 25 }
];

downloadExcel(data, '파일명', '시트명');
```

### 3. 엑셀 다운로드 핸들러
```javascript
const handleExcelDownload = useCallback(async () => {
  try {
    const hasSelection = selectedRows.size > 0 || allSelectedRows.size > 0;
    
    if (hasSelection) {
      // 선택된 항목만 다운로드
      let allSelectedData = [];
      for (const [pageKey, pageData] of allSelectedRows) {
        allSelectedData = allSelectedData.concat(pageData);
      }
      const excelData = allSelectedData.map((row, index) => ({
        'No.': index + 1,
        // ... 데이터 매핑
      }));
      downloadExcel(excelData, '파일명_선택항목', '시트명', true, toast);
    } else {
      // 전체 데이터 다운로드 (새로운 API 사용)
      setLoading(true);
      
      // 새로운 엑셀 다운로드 API 사용
      const countResponse = await excelDownloadMemberAccount(0, 1);
      const totalCount = countResponse.data.totalElements || 0;
      
      if (totalCount <= 50000) {
        // 5만개 이하면 한 번에 받기
        const response = await excelDownloadMemberAccount(0, totalCount);
        const data = response.data;
        if (data.content) {
          const allData = data.content.map((item, index) => ({
            'No.': index + 1,
            // ... 데이터 매핑
          }));
          downloadExcel(allData, '파일명_전체', '시트명', true, toast);
        }
      } else {
        // 5만개 초과면 청크 단위로 받기
        const chunkSize = 50000;
        const totalChunks = Math.ceil(totalCount / chunkSize);
        let allData = [];
        
        for (let i = 0; i < totalChunks; i++) {
          const chunkResponse = await excelDownloadMemberAccount(i, chunkSize);
          const chunkData = chunkResponse.data;
          if (chunkData.content) {
            const chunkExcelData = chunkData.content.map((item, index) => ({
              'No.': allData.length + index + 1,
              // ... 데이터 매핑
            }));
            allData = allData.concat(chunkExcelData);
          }
          console.log(`엑셀 다운로드 진행률: ${i + 1}/${totalChunks}`);
        }
        downloadExcel(allData, '파일명_전체', '시트명', true, toast);
      }
      setLoading(false);
    }
  } catch (error) {
    console.error('엑셀 다운로드 오류:', error);
    toast.error('엑셀 다운로드 중 오류가 발생했습니다.');
    setLoading(false);
  }
}, [selectedRows, allSelectedRows, toast]);
```

## 새로운 엑셀 다운로드 API 엔드포인트

### 1. 회원 자산 내역
```javascript
import { excelDownloadMemberAccount } from '../../../../api/auth/JihunAuth';

// 사용법
const response = await excelDownloadMemberAccount(page, size);
```

### 2. 회원 자산 현황
```javascript
import { excelDownloadMemberAssetDetails } from '../../../../api/auth/JihunAuth';

// 사용법
const response = await excelDownloadMemberAssetDetails(page, size);
```

### API 특징
- 기본 size: 50,000개
- 최대 size: 100,000개
- EntityNotFoundException 방지
- 메모리 효율적 처리
- 엑셀 다운로드에 최적화된 응답 형식

### 응답 형식
```json
{
  "content": [...],
  "totalElements": 350000,
  "totalPages": 7,
  "currentPage": 0,
  "size": 50000,
  "hasNext": true,
  "hasPrevious": false
}
```

## 주의사항
1. 대용량 데이터 다운로드 시 청크 단위 처리 권장
2. 네트워크 안정성을 위해 5만개씩 처리
3. 선택된 항목이 있으면 선택된 항목만 다운로드
4. 선택된 항목이 없으면 전체 데이터 다운로드
5. 새로운 API는 EntityNotFoundException을 방지하도록 설계됨
